# Corsac-engine

C++, SDL2, OpenGL - Game engine.

SDL2 2.0.10(stable);
OpenGL 4.6 NVIDIA

# Install

```bash

sudo apt install libsdl2{,-image,-mixer,-ttf,-gfx}-dev

sudo apt-get install libglew-dev

git clone https://github.com/Survereignty/Corsac-engine.git

cd Corsac-engine

# build and run
make

```
